Measuring Units

Measuring units plays a very important role in styling a webpage. All these units mentioned below are commonly used by the developer. Where there are allot of Css measuring units

Px - Pixels

The are the most fundamental units to measure a websites elements. An it is widely used as a base units and other as over it. All the other units like em and rem work only when px units are declared as base size. Px is used to give a size factor to the font, container width and in every html element sizing unit is must.

This div has a width and height of 50px

This font has 30px of the font-size

% - percentage

This percentage unit is used when we are trying to give sizing based on the parents container width.

The width of this container is 50% percent of its parent. That is the parent width is 300px and the child width will be 150px

Percentage in font is not widely used but it set the font-size to the 100% of the base-font-size of the page or its parent. Now the font-size of its parent is set to 30px and the font-size of this is set to 50% that is 50px.

em units

Em units are pretty much similar to the percentage unit but it adds the size based on the parents font size and it don't inherit the page font-size. Here 1em is = entire font size. Example 1em of 10px is 10px. This unit can be used to create a relative environment in a container. For Example: Styling buttons when the border-radius should be much relative to width when the width is changed even the border-radius must be changed.

The width of this container is 15em .That is the parent 15x size of the parents font-size. this parents font-size is set to 10px then the width will be 50px

The font-size is set to 1.5em that is 15px. Based on the parent-font-size all the child elements can be sized

Rem units

Rem units are the units used to style the entire pages elements based on one default font-size. The font size is declared in the html query styling and that font-size is set as default font size of that page. Example if the html font size is 10px then 1rem unit will be 10px no matter in which container you get the same size. These units can be used in any Css property following border, padding, margin, border-radius and much more.

The width of this container is 30rem that is 300px. The page font-size is set to 10px. 30X times of the base font-size

The font-size is set to 2rem that is 20px.
